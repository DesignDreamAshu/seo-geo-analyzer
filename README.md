# SEO & GEO Analyzer Monorepo\n\nThis repository hosts the frontend (Vite + React) and backend (Express + Lighthouse) applications using npm workspaces.\n\n## Getting Started\n\n1. Install all dependencies from the repo root:\n\n`ash\nnpm install\n`\n\n2. Start both apps with a single command:\n\n`ash\nnpm run dev\n`\n\n- Frontend: http://localhost:8080\n- Backend API: http://localhost:4000\n\n### Workspace scripts\n\nYou can run each workspace individually if needed:\n\n`ash\n# Frontend only\nnpm --workspace apps/frontend run dev\n\n# Backend only\nnpm --workspace apps/backend run dev\n`\n\nThe backend streams Lighthouse audits and handles export/share endpoints; the frontend consumes those APIs and renders the dashboard.\n\n## Structure\n\n``\nseo-geo-analyzer/\n+- apps/\n¦  +- frontend/      # React + Vite UI\n¦  +- backend/       # Express API\n+- packages/\n¦  +- shared/        # Shared utilities/types (placeholder)\n+- README.md\n+- package.json      # npm workspaces + concurrent scripts\n+- .gitignore\n``\n
